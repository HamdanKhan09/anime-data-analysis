# Anime Dataset Analysis

Exploratory data analysis on top-ranked anime data.

## What this project does
- Extracts features (episodes, start date, end date) from raw messy text data using custom Python functions
- Calculates total broadcast duration for each anime
- Visualizes insights through 4 charts

## Visualizations
- Top 10 Anime by Score
- Distribution of Episodes
- Score vs Episodes relationship
- Anime releases by year

## Tools
Python | Pandas | Matplotlib | Jupyter Notebook
